# Supply Chain Analytics Dashboard — Power BI

## Overview
End-to-end Power BI project analysing 180,519 supply chain orders 
across 5 global markets (2015–2018).

## Key Findings
1. 54.8% late delivery rate — systemic issue across all shipping modes
2. Europe leads revenue at $97M but Pacific Asia has lowest margin (11.5%)
3. Fishing category generates $6.9M — 3x the next highest category
4. Consumer segment drives 52% of total revenue
5. Average 3.18 orders per customer — retention opportunity identified

## Technical Stack
- Power BI Desktop (Data modelling, DAX, Visualisation)
- Power Query (ETL, data cleaning, star schema creation)
- 25+ DAX measures including time intelligence (YoY, MoM, YTD, Rolling 3M), delivery KPIs, customer segmentation, and discount impact analysis

## Data Model
Star schema — 1 fact table + 6 dimension tables — 180,519 rows
